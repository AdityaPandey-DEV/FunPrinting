# Frontend Design Plugin

Create distinctive, production-grade frontend interfaces that avoid generic AI aesthetics.

## What it does

This plugin provides a `frontend-design` skill that is automatically invoked when building web components, pages, or applications. It guides Claude to make bold, intentional design choices rather than defaulting to generic "AI slop" aesthetics.

## Key Principles

- **Bold Aesthetic Direction**: Choose and commit to a clear design direction (brutalist, maximalist, minimalist, retro-futuristic, etc.)
- **Distinctive Typography**: Use unique, characterful fonts instead of generic choices like Inter or Roboto
- **Cohesive Color Palettes**: Dominant colors with sharp accents, using CSS variables for consistency
- **Purposeful Motion**: High-impact animations and micro-interactions
- **Creative Layouts**: Unexpected compositions, asymmetry, overlap, and grid-breaking elements

## Anti-Patterns (What to Avoid)

- Generic purple gradients on white backgrounds
- Overused font families (Inter, Roboto, Arial, system fonts)
- Predictable layouts and component patterns
- Cookie-cutter designs lacking context-specific character
